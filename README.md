# Controle Financeiro · Família da Luz

App PWA completo para controle financeiro pessoal e empresarial.

## Como usar

### Pré-requisitos
- [Node.js](https://nodejs.org/) v18 ou superior

### Instalação

```bash
# Na pasta do projeto:
npm install

# Iniciar em modo desenvolvimento:
npm run dev
```

Acesse em: **http://localhost:5173**

### Build para produção
```bash
npm run build
npm run preview
```

## Funcionalidades
- **Finanças Pessoais**: Entradas, saídas por categoria, saldo mensal, gráficos
- **Empresa**: Produtos, vendas, controle de estoque, relatório de lucro
- **Lembretes**: Contas a pagar/receber com alertas visuais
- **PWA**: Instale no celular como app nativo
- **Backup**: Exporte e importe dados como JSON
- **Offline**: Funciona sem internet após instalação
